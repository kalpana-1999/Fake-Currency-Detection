Introduction

Counterfeit currency circulation is a significant threat to the financial stability and economic structure of any nation. With advancements in printing technology, fake notes have become increasingly sophisticated, making manual detection unreliable.

Traditional techniques such as ultraviolet scanning, watermark verification, and physical inspection require trained personnel and specialized equipment. These methods are time-consuming and prone to human error.

This project proposes an intelligent, automated fake currency detection system using machine learning and computer vision techniques to accurately classify currency notes as genuine or fake.

Machine Learning

Uses CNN models to automatically learn complex visual patterns from currency note images.

Image Processing

Analyzes security features such as textures, patterns, and print quality using image processing techniques.

Real-Time Detection

Provides instant results with confidence score, suitable for banking and retail environments.
